# HR Portal

Простий HR Portal додаток на React + Vite + Tailwind CSS

## Запуск

```bash
npm run dev      # Запустити dev сервер
npm run build    # Зібрати для продакшену
npm run preview  # Переглянути білд
npm run lint     # Перевірити типи TypeScript
```

## Структура

```
src/
├── app/        # React компоненти
├── styles/     # CSS файли
└── main.tsx    # Вхідна точка
```
